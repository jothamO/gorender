# Support Policy (Draft)

## Where To Ask

- Bug reports: GitHub Issues (after launch).
- Usage questions: Discussions (after launch).
- Security concerns: private security channel (see `SECURITY.md`).

## What To Include

- command used
- logs/errors
- OS + Go + Chrome + ffmpeg versions
- minimal reproduction URL/composition

## Support Boundaries

- Core CLI/runtime is supported first.
- Optional modules may have best-effort support.
